# CareerByte AI 🚀

**Your Personal AI Career Copilot**

CareerByte AI is an AI-powered career management platform that helps job seekers discover opportunities, optimize resumes, track applications, prepare for interviews, and improve their chances of landing offers.

## ✨ Features

### 📄 Resume Intelligence

* Resume Upload & Management
* PDF/DOCX Parsing
* Skill Extraction
* ATS Score Analysis
* Resume Versioning
* AI-Powered Resume Tailoring

### 🔍 Job Discovery

* Remote Job Aggregation
* Greenhouse Integration
* Lever Integration
* Ashby Integration
* RemoteOK Integration
* We Work Remotely Integration
* Job Deduplication & Normalization

### 🤖 AI Match Engine

* Resume-to-Job Matching
* ATS Compatibility Scoring
* Skill Gap Analysis
* Priority Ranking
* Personalized Recommendations

### 📝 Application Assistant

* Tailored Resume Generation
* Cover Letter Generation
* Recruiter Outreach Messages
* Application Preparation

### 📊 Application Tracker

Track your entire hiring journey:

* New
* Saved
* Ready to Apply
* Applied
* Online Assessment
* Interview
* Offer
* Rejected

### 📈 Analytics Dashboard

* Jobs Discovered
* Match Rate
* Applications Sent
* Interview Rate
* Offer Rate
* Hiring Funnel Analytics

### 📅 Productivity Integrations

* Gmail Integration
* Google Calendar Integration
* Telegram Notifications
* Follow-Up Reminders

## 🚀 Getting Started

### Clone Repository

```bash
git clone https://github.com/karthikrshet/CareerByte-AI.git
cd CareerByte-AI
```

### Install Dependencies

```bash
npm install
```

### Configure Environment Variables

Create a `.env` file:

```env
DATABASE_URL=
NEXTAUTH_SECRET=
NEXTAUTH_URL=
GEMINI_API_KEY=
REDIS_URL=
```

### Run Database Migrations

```bash
npx prisma migrate dev
```

### Start Development Server

```bash
npm run dev
```

Open:

```text
http://localhost:3000
```
